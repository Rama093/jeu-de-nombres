# 🎯 Jeu des Nombres

Petit jeu en JavaScript pour 2 joueurs. 

Chaque joueur doit cliquer sur les nombres dans l’ordre croissant.  
Le joueur qui entoure le plus de nombres gagne !


## Technologies utilisées

- HTML5  
- CSS3  
- JavaScript
